# Tutorial

## [Orthoformer Practical Tutorial for Biologists](./Biologist_Tutorial.md)
Practical, step-by-step guide for microbiologists and ecologists to use Orthoformer without deep ML background. Covers installation, model download, OG-based input preparation, genome embedding extraction via [biologist_quickstart.py](./biologist_quickstart.py), and example downstream analyses (PCA/UMAP, phylogenetic trees, taxonomy). Includes troubleshooting and a quick-reference command card.

## [MAG Workflow Tutorial: From Genes to Applications](./MAG_Tutorial.md)
End-to-end MAG workflow from protein-coding genes to Orthoformer applications. Walks through gene calling (Prodigal), eggNOG-mapper annotation, [mag_preprocess.py](./mag_preprocess.py) conversion to a tokenized dataset, embedding extraction, and downstream use cases (comparison with reference genomes, functional trees, taxonomic placement). Complements the general biologist tutorial with MAG-specific QC notes and batch-processing guidance.
